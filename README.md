# LemonUI 柠檬UI

这是一款基于Vue3.0的轻量级UI组件库。简单清爽的风格希望你会喜欢

## 安装

执行语句

```cmd
npm install lemon-design -S
```

## 使用

全局使用请在您的main.js中增加如下语句

```cmd
import { createApp } from 'vue'
import App from './App.vue'
import LemonUI from 'lemon-design'

const app = createApp(App)
app.use(LemonUI)
app.mount('#app')
```
